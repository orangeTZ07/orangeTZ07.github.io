本文介绍欧润吉在高数学习中对空间旋转曲面方程的一点理解

本文也算是一点学习笔记吧，不想记到本子上而已。

首先是空间曲面如何书写公式。

空间向量是依赖于二维平面进行生成的，其生成方式有很多，这里我们只聚焦于二维平面旋转所引起的生成。
首先做个示例：一个xOy上的圆围绕x轴旋转成为圆球。假设这个圆是 x平方 + y平方 = 4 现在它要旋转成空间球体，它将在z轴上留下足迹。在实际题目中，可能会这样描述 大括号{ x平方 + y平方 =4 \n z = 0 ， 求它的旋转体的方程式 这种情况下我们可以先把x给固定住，不修改x，然后把z加到y上 x平方 + 1*(y平方 + z平方) = 4 这时候我们就得到了球体的方程。

这个过程实际上是对空间曲面截痕的一次旋转。什么是截痕法？简单来说比如你有一个椭球体（假设就是一个xOy椭圆所形成的） x平方/a平方 + y平方/b平方 + z平方/c平方 = 1 在截痕法中，假设我们目前不容易接受加上z轴的空间曲面，那么我们可以直接把z设置为0。这时候相当于直接平行于xOy对着椭球体切了一刀，刚好切到z = 0的位置。 这时候变成了什么？x平方/a平方 + y平方/b平方 = 1 是不是很熟悉了？这就是个二维平面上的椭圆。 然后我们再对上述求球体方程的方法如法炮制: 我们首先按要求，把椭圆绕x轴旋转，然后把z加到y上(绕其他轴也可以，绕哪个就给哪个加)。得到空间曲面方程： x平方/a平方 + (y平方 + z平方)/b平方 = 1

你可能会问，为什么一开始是x平方/a平方 + y平方/b平方 + z平方/c平方 = 1,后面怎么就变成x平方/a平方 + (y平方 + z平方)/b平方 = 1了？c怎么变成b了？ 其实这里是因为我们是去拿着z=0的截痕，也就是那个椭圆去旋转的。椭圆在y轴正半轴上的长度是b，所以它绕着x轴旋转的时候，打到z轴上的长度也是b。所以在这次旋转中c = b。 那我们遇到c != b的情况是怎么回事？ 这种通常是旋转并构造了空间曲面方程后对某个轴的长度进行了伸缩。比如我们要在z轴上对刚刚的椭圆体进行压扁，把它压扁到原来的2/3，我们可以对z乘3/2（因为z轴数值要增长得更快才能变扁嘛），此时得到： x平方/a平方 + y平方/b平方 + (z * 3/2)平方/b平方。此时c = 2/3 * b

那么有没有更多例子？
有的，我们上点类型和前边那俩大不相同的。

  • 椭圆锥
    我们先构造一条直线y = kx，也就是y平方 = k平方*x平方，也就是y平方 = j * x平方。 然后我们把直线绕y轴旋转，按照上述方法添加上z轴（旋转到z轴区域了） y平方 = j * (x平方 * z平方) 此时我们得到了圆锥体 然后我们按照上述压缩方法对不同轴的长度进行压缩即可得到椭圆椎。 然后我们就得到了一个z轴方向平放的无限长的扁平哑铃。
  • 单叶双曲面
    我们首先构造双曲线 y平方/a平方 - z平方/b平方 = 1 然后把双曲线绕z轴旋转，旋转到了x轴的领域。按照上述方法，我们要把x加到y上，得到了： (y平方 + x平方)/a平方 - z平方/b平方 = 1

以上就是我在学习空间曲面旋转体时总结的方法和经验。
